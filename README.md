# Covid19-Tracker
Created a Covid19 Tracker App using Bulma, Flask and SQLite. It also has feature to add Patient Details.

## Installation
Use the requirements.txt file to install required libraries.
```bash
pip install -r requirements.txt
```

## Usage
To run the Project
```bash
python app.py
```
